# 🔮 전생 리더기

> 이름을 입력하면 당신의 전생을 알려드립니다!

기원전 15만 년부터 1980년까지, 랜덤한 시대와 직업(혹은 존재)을 부여받고 OpenAI가 당신만의 전생 스토리를 만들어줍니다.

![Express](https://img.shields.io/badge/Express-4.x-000000?logo=express)

## 📋 주요 기능

- **전생 스토리텔링** — 이름 기반으로 AI가 전생 이야기를 생성
- **300개 이상의 직업/존재** — 인간, 동물, 무생물, 미생물, 외계인, 판타지, 고대생물, 초자연 존재
- **100개 이상의 사망 원인** — 진지한 것부터 황당한 것까지
- **40개 역사 시대** — 고조선, 삼국시대, 고려, 조선부터 로마, 바이킹, 디스코 시대까지
- **공유 & 복사** — 결과를 친구에게 바로 공유

## 🚀 설치 및 실행

### 1. 클론

```bash
git clone git@github.com:tiaspalding13082-star/deploy-test1.git
cd deploy-test1
```

### 2. 의존성 설치

```bash
npm install
```

### 3. 환경변수 설정

프로젝트 루트에 `.env` 파일을 생성하고 OpenAI API 키를 입력합니다:

```
OPENAI_API_KEY=sk-proj-여기에_본인의_API키_입력
```

> ⚠️ `.env` 파일은 `.gitignore`에 포함되어 있어 GitHub에 업로드되지 않습니다.

### 4. 서버 실행

```bash
npm start
```

브라우저에서 **http://localhost:3000** 접속

## 🎮 사용법

1. 메인 화면에서 **이름을 입력**합니다
2. **"전생 보기"** 버튼을 클릭합니다
3. AI가 전생 스토리를 생성하는 동안 잠시 기다립니다
4. 결과 확인:
   - 📅 **시대** — 어느 시대에 살았는지
   - 🗓️ **년도** — 구체적인 연도
   - 👤 **존재/직업** — 전생에서 무엇이었는지
   - 💀 **사인** — 어떻게 생을 마감했는지
   - 📖 **스토리** — AI가 만든 전생 이야기
5. **공유하기** / **복사하기**로 친구에게 보내기
6. **다시 보기**로 다른 전생도 확인 가능

## 🌟 응용 예시

### 모임/파티 아이스브레이커
참가자들이 각자 이름을 입력하고 전생 결과를 발표하며 웃음을 유발합니다. "나 전생에 조선시대 엿장수였대!" 같은 대화가 자연스럽게 이어집니다.

### SNS 바이럴 콘텐츠
공유하기 기능을 활용해 인스타그램 스토리, 트위터, 카카오톡 등에 결과를 공유합니다. "내 전생 알아보기" 링크와 함께 확산시키면 자연스러운 바이럴 효과를 기대할 수 있습니다.

### 커플/친구 궁합 놀이
두 사람이 각각 전생을 확인한 뒤, 같은 시대에 살았는지, 서로 만났을 가능성이 있는지 상상하며 이야기를 만들어봅니다.

### 학교/학원 역사 수업 보조 도구
학생들이 랜덤으로 받은 시대와 직업을 바탕으로 해당 시대를 조사하고 발표하는 활동에 활용할 수 있습니다.

### 콘텐츠 크리에이터 소재
유튜브, 틱톡 등에서 "전생 리더기 결과 리액션" 영상을 제작하거나, 시청자 이름으로 전생을 읽어주는 라이브 방송에 활용할 수 있습니다.
